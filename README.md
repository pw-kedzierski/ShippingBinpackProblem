# Bin Packing App

Blazor WebAssembly aplikacja do ręcznego pakowania statków w kotwicowisku.

## Wymagania

- .NET 10 SDK

## Uruchomienie

Najpierw uruchom API proxy:
```bash
dotnet run --project Api/BinPackingApp.Api.csproj
```

Następnie uruchom aplikację Blazor:
```bash
dotnet run
```

## Testy

```bash
dotnet test
```
